CW-adapter-fs.js:
javascript(function() {
  // entire file content here
  globalThis.Adapters.fs = { init, select, create, update, delete: del, reindex };
  console.log("✅ CW-adapter-fs.js loaded");
})();
pb-adapter-pocketbase.js:
javascript(function() {
  // entire file content here
  globalThis.Adapters.pocketbase = { init, select, create, update, delete: del };
  console.log("✅ pb-adapter-pocketbase.js loaded");
})();
All internal function names become local to the IIFE scope. No global pollution. No cross-adapter overwrites. The only globals set are Adapters.fs and Adapters.pocketbase — intentional.
